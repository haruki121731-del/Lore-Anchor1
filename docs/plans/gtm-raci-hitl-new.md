# Lore-Anchor 新RACI（AI/自動化＋HITL再設計）

## 判定前提

- Level 0: 完全自動化（不可逆性が低く、ルール化が完全）
- Level 1: 例外時のみ人間介入（確信度低下/異常時）
- Level 2: 実行前に人間承認が必須（外部発信/システム変更/不可逆アクション）
- Level 3: 人間主体で実行、AIは補助（高度文脈/関係構築/最終責任）

運用閾値（共通）:
- `AI confidence < 0.80` は自動的に人間レビューへエスカレーション
- `policy_violation_flag = true` は実行停止して承認待ち
- `anomaly_score > 0.70` は例外処理キューへ送る
